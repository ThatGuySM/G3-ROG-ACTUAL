Operational Notes / Gotchas

  • The uploaded AIDA64 reports are now the source of truth for the installed build. The older g3wactual spec sheet still listed the 4090/B650E/PCCOOLER setup and should be considered stale for this machine.
  • CPU is listed as Ryzen 7 9850X3D. AMD's current product listing confirms that model exists, so this is not just an AIDA typo from old 9800X3D naming.
  • Memory is running at DDR5-6200 with 32-38-38-80 timing. That is spicy but not automatically bad. Stability testing decides whether it stays king or gets demoted.
  • X870E Dark Hero M.2_2 shares bandwidth with the rear USB4 40Gbps ports. Avoid filling M.2_2 unless you intentionally accept that tradeoff.
  • Use the proper native high-power GPU cable path. No splitter shenanigans on an RTX 5090. That is how expensive smoke becomes a home fragrance.
  • Keep both AIDA64 source reports with this HTML/PDF. They are the receipts for what Windows actually detected on 2026-05-08.
